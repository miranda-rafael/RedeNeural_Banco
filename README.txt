# RedeNeural-Banco

aplicação de uma rede neural artificial a partir de um dataset bancário

##Motivação

Queremos entender o motivo de nossa agência  bancária estar com uma alta taxa de evacuação dos clientes. Para isso construimos uma ANN (rede neural artificial) que nos mostrará o comportamento de dez mil clientes, aleatóriamente selecionados e a partir disso, sempre que um relatório ou perfil chegar, iremos prever se o cliente irá ou não nos deixar.
